Installation Back to top

  1. First, make sure you are using valid DOCTYPE
  2. Include the jQuery library, the slider CSS file and the slider javascript in the head of the page(s) where you want to use the slider. 
    <script type="text/javascript" src="javascript/jquery.min.js"></script>

<link rel="stylesheet" type="text/css" href="javascript/slider/themes/default/jquery.slider.css" />
<!--[if IE 6]>
<link rel="stylesheet" type="text/css" href="javascript/slider/themes/default/jquery.slider.ie6.css" />
<![endif]-->

<script type="text/javascript" src="javascript/slider/jquery.slider.min.js"></script>

    Where default is the theme that you have chosen, and can be:

    • default
    • carbono
    • fresh
    • minimalist
  3. Then, initialize the plugin. Put the following code before the closing tag of your body 
    <script type="text/javascript">
  jQuery(document).ready(function($) {
    $(".myslider").slideshow({
      width      : 900,
      height     : 360
    });
  });
</script>
  4. The above code will take all images from the div 'myslider' and it will create the slideshow. Here is how it looks like that div: 
    <div class="myslider">
  <div>
    <img src="assets/photo_1.jpg" alt=""/>
  </div>
  <div>
    <img src="assets/photo_2.jpg" alt=""/>
  </div>
  <div>
    <img src="assets/photo_3.jpg" alt=""/>
  </div>
</div>
  5. If you want a specific transition for each image put the class "jquery-slider-trans-ANIMATION" for each div where ANIMATION is the effect that you want. For example: 
    <div class="slider">
  <div class="jquery-slider-trans-explode">
    <img src="assets/photo_1.jpg" alt=""/>
  </div>
  <div class="jquery-slider-trans-fade">
    <img src="assets/photo_2.jpg" alt=""/>
  </div>
  <div class="jquery-slider-trans-squarerandom">
    <img src="assets/photo_3.jpg" alt=""/>
  </div>
</div>
  6. If you want the same transition for all images, add the option on the initialization: 
    <script type="text/javascript">
  jQuery(document).ready(function($) {
    $(".slider").slideshow({
      width      : 900,
      height     : 360,
      transition : "explode"
    });
  });
</script>

    Where transition can be:

    • fade
    • bar
    • barleft
    • barright
    • square
    • squarerandom
    • slide
    • slideleft
    • slideright
    • slidetop
    • slidebottom
    • fountain
    • rain
    • explode
    • exploderandom

    See a live example of all transitions here.

  7. These are the options (with their default values) that you can customize on the initialization: 
    navigation      : true, // Show navigation buttons (« »)
selector        : true, // Show selector buttons (1 2 3 4 5)
timer           : true, // Show timer (it may be different in each theme)
control         : true, // Show control button (Play/Pause)

pauseOnClick    : true, // Pause the slideshow when you click any button
pauseOnHover    : true, // Pause the slideshow when you hover the slider
loop            : true, // Slideshow will start from the beginning when the end is reached
slideshow       : true, // Start the slideshow when page loads

width           : 500,
height          : 332,

delay           : 4500, // in ms. Time between slides
transition      : 'random',

duration        : 400 // in ms. Speed of the animation
bars            : 9, // Number of bars for "bar, fountain, rain"
columns         : 7, // Number of columns for "square, explode"
rows            : 3, // Number of rows for "square, explode"
padding         : 8  // Width of padding in px for "explode"

Support Back to top

This plugin supports the following browsers:

  • Internet Explorer 6+
  • Firefox 2 +
  • Safari 3 +
  • Chrome 3 +
  • Opera 9 +

Quirks mode is not supported in any browser, but that doesn't necessarily mean that it won't work. This means that you need to use either a strict or transitional DOCTYPE.

Frequently Asked Questions Back to top

  1. Can I use this script on my client's website?

    Yes. Your license includes the rights to use the script on any one website at a time. If you'd like to use the script as part of a service, please consider purchasing an Extended License.

  2. Can I make modifications to this script?

    Feel free to make any modifications you need. Also feel free to blog and show it to your friends, but please do not redistribute the script with your changes.

  3. Why the script doesn't work?

    Make sure you don't have multiple versions of jquery in your page. Look at the source view in your browser to see what javascript files you are loading.

    For example:

    <script src="crazy_path/jquery.js" type="text/javascript"></script>
...
<script src="crazy_path/jquery.min.js" type="text/javascript"></script>
...
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js"></script>

    jquery.js and jquery.min.js are the same script. Delete the last one if you can. Keep only one.

  4. The slider is positioned incorrectly or behaving strangely in Internet Explorer

    This is likely a doctype issue. This plugin requires a valid doctype and rendering in quirks mode is not supported. Make sure you are using the full doctype declaration to insure rendering in standards mode.

    This abbreviated doctype renders the document in quirks mode for Internet Explorer:

    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">

    The doctype with URI renders in standards mode for all browsers:

    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
